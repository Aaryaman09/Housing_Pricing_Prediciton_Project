## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

create environment

```bash
  conda create -p venv python==3.7 -y
```

activate environment

```bash
  conda activate venv/
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  python app.py
```
